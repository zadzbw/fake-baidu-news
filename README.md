# fake-baidu-news

## 项目概述

1. 此项目为百度新闻移动端的模仿版本。
2. 所用技术框架为 [React][1] + [Redux][2] + [ES6][3] + [Less][4] + [webpack][5]。
3. 此项目主要目的是自己练习，增强自身技术水平。

### 项目启动

安装依赖：

```
npm install
```

开发模式运行：`localhost:9999`

```
npm start
```

打包编译：

```
npm run build
```
